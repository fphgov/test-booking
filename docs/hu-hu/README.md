# Dokumentáció

A Főváros Önkormányzata által létrehozott, koronavírus gyorstesztelés menedzseléséhez használt backend alkalmazás telepítési útmutatója. Az útmutató a lehető legbővebb tájékoztatást probál adni a követelményrendszerrel, telepítéssel, konfigurálással és üzemeltetéssel kapcsolatban, ennek ellenére elképzelhető, hogy egyes helyeken hiányosak lehetnek. A tájékoztató a saját ajánlásainkat, preferenciáinkat tartalmazzák, ettől sok esetben ellehet térni.

Az útmutató nem helyetesíti a fejlesztői és üzemeltetői (DevOps) tudást, ugyanakkor egy jó közelítést add a lehetséges üzemeltetési formára és adott esetben a projekt tovább fejlesztésre. Nem minden esetben térünk ki a legbiztonságosabb megoldásokra, ugyanakkor erősen ajánlottak a biztonsági megoldások használata, mint például `fail2ban`, `ufw`, `ModSecurity` vagy egyéb WAF használata.
